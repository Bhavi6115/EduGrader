## 📖 Overview

EduGrader is an AI-powered grading assistant that turns student submissions into **instant, actionable feedback** in under 2 seconds. 

Teachers simply paste the student's work and a rubric, and EduGrader outputs:
- ✅ **Strengths** (What the student did well)
- ⚠️ **Areas for Improvement** (Where they lost marks)
- 🚀 **Actionable Next Steps** (Specific, sentence-level feedback)

All feedback can be **copied**, **downloaded as TXT**, or **exported as a professional PDF report**.

## ✨ Features

| Feature | Description |
| :--- | :--- |
| 🤖 **AI-Powered Grading** | Uses Groq's Llama 3.1 (free & lightning fast) to analyze submissions. |
| 📝 **One-Click Sample Essay** | Instantly loads a sample essay + rubric for quick demos. |
| 🎨 **Visual Feedback Dashboard** | Color-coded pills for Strengths (Green) & Weaknesses (Red). |
| 🚀 **Actionable Next Steps** | Clear, sentence-by-sentence feedback in beautiful blue cards. |
| 📄 **TXT Export** | Download the entire feedback report as a `.txt` file. |
| 📑 **PDF Export** | Generate a professional, printable PDF report for students. |
| 📋 **Copy to Clipboard** | Select and copy the feedback with one click (or using Ctrl+A). |
| 🔒 **Secure** | API keys stored securely in Streamlit Cloud Secrets — never exposed. |
